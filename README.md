Lifestyle
=========

Naive attempt at household management, with movie watchlists,
shopping lists, holiday planning, etc.

needs:
- Phalcon
- Mongo

Tested with:
- Ubuntu 16.04
- Apache 2.4.18
- PHP 7.0
- Phalcon 3.1
- php-mongodb 1.1.5
- Phalcon Incubator branch 3.1
